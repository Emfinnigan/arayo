브랜드 이름

왜 "아라요"인가

"아라요(알아요)"는 한국어로 "나는 알고 있어요"라는 뜻입니다. K-드라마의 거의 모든 대화, K-POP의 거의 모든 가사에서 들을 수 있는, 전 세계적으로 가장 많이 인식되는 한국어 표현 중 하나입니다. 우리의 브랜드 약속을 완벽하게 담고 있습니다: 우리는 한국을 알고 있으니까, 당신은 고민하지 않아도 됩니다.

더 깊은 문화적 의미도 있습니다. "난 알아요"는 1992년 서태지와 아이들의 데뷔곡으로, K-POP의 시작이자 한류 열풍의 기점이 된 노래입니다. 한국 문화가 세계로 뻗어나간 그 순간의 DNA를 담은 이름, 그것이 아라요입니다.

해결하는 문제

64%가 음식 때문에 옵니다. 그런데 현장 경험이 엉망입니다.

진짜 맛집을 못 찾습니다

30년 된 골목 곱창집, 간판 없는 할머니 김치찌개집 — 영어 리뷰도 없고, 구글에도 안 나오고, 밖에서 보면 들어가기 무섭습니다.

메뉴를 읽을 수 없습니다

코팅된 한국어 메뉴판. 구글 번역기는 "돼지 내장 볶음"이라고 알려주지만, 실제로 어떤 맛인지, 어떻게 나오는지, 왜 주문해야 하는지는 전혀 모릅니다.

밤나들이 문화를 모릅니다

한국인은 1차→2차→3차 (식사 → 술집/포차 → 야식)로 이동합니다. 이 전체 문화적 경험이 외국인에게는 보이지 않습니다.

제품 기능

하나의 경험으로 작동하는 네 가지 기능

🔍 큐레이션 맛집 발견

100개 이상 직접 방문하여 검증한 숨은 맛집. 영상/릴스, 실제로 먹어본 사람이 쓴 영어 설명, 동네 맥락, "이 가게가 특별한 이유." 음식 종류, 동네, 가격, 분위기별 필터링 가능.

🌙 밤나들이 플래너 (1차→2차→3차)

킬러 기능. 동네와 분위기를 고르면 완전한 한국식 밤나들이를 만들어줍니다: 식사 → 술 → 야식, 도보 경로와 예상 시간, 예산 포함. "홍대에서 5만원 이하로 밤나들이 짜줘."

📋 스마트 메뉴 번역

단순 번역이 아닙니다. 각 메뉴 항목에: 실제로 뭔지, 어떤 맛인지, 어떻게 나오는지, 매운 정도, 추천 여부를 제공합니다.

🤖 AI 다이닝 동반자 (향후)

채팅 기반 AI가 한국인 친구처럼 도와줍니다. "BBQ집인데 3명이서 뭘 시켜야 할지 모르겠어요" → 부위를 추천하고, 굽는 법을 알려주고, 소스를 설명합니다.

경쟁 우위: 직접 방문하여 검증한 100개 맛집 + 번역된 메뉴 + 영상 콘텐츠. 이것은 크롤링이나 AI로 복제할 수 없는 발품입니다. 큐레이션 자체가 제품입니다.

해결하는 문제

K-드라마가 욕망을 만듭니다. 아무것도 그것을 탐험으로 전환하지 않습니다.

촬영지가 흩어져 있습니다

사랑의 불시착 촬영지를 방문하고 싶은 관광객 — 서울, 강원도 등에 흩어져 있습니다. 드라마별로 그룹화하고, 어떤 곳이 방문할 가치가 있는지 알려주는 앱이 없습니다.

92%가 서울에만 머뭅니다

정부는 지방 관광 활성화에 수십억을 쓰고 있습니다. "강원도를 방문하세요"는 매력적이지 않습니다. "도깨비 퀘스트를 완료하고 주문진 해변 체크포인트를 잠금 해제하세요"는 매력적입니다.

체크리스트 너머로 갈 이유가 없습니다

대부분 경복궁 → 명동 → N서울타워를 하고 떠납니다. 진행 시스템도, 수집 메카닉도, 더 깊이 탐험할 이유도 없습니다.

제품 기능

포켓몬 GO와 한국 문화의 만남

📍 퀘스트 시스템

각 K-드라마가 "퀘스트"가 됩니다 — 3-8개 촬영지의 큐레이션된 트레일. 스토리 맥락, 장면 스크린샷, 사진 매칭 챌린지, 트리비아 포함. 모든 GPS 체크포인트를 완료하면 퀘스트 배지 획득.

📸 사진 매칭 & 리더보드

드라마의 정확한 장면을 보여주고 같은 각도로 재현하도록 도전. K-Quest 워터마크와 함께 인스타그램에 공유 — 무료 바이럴 마케팅. 주간/월간 랭킹으로 새로운 지역 탐험 유도.

🗺️ K-드라마를 넘어서

K-POP 트레일 (BTS 성지순례), 역사 유적 (조선왕조 트레일), 음식 퀘스트 ("종로 5대 찌개 완주"), 시즌 트레일 (벚꽃, 단풍). 각 카테고리가 새로운 사용자층을 가져옵니다.

경쟁 우위: 100개 이상 직접 방문하여 GPS 확인된 위치 + 드라마별 콘텐츠 + 정확한 사진 각도. 이것은 애그리게이터가 복제할 수 없는 현장 작업입니다.
